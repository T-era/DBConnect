DBConnect
=========

Java, databse client using jdbc.

Swing GUIによる、JDBCクライアントツールです。
Javaプログラミング時に、JDBCのレスポンスを素のままで観察したかったことから作成を始め、あれこれ機能追加をしてたらこうなりました。
以下のように思う方は使ってみてくださいな。

・ DBMSによってクライアントツール違うのは、キーバインドなどが違って不便！
・ 正規のクライアントツールが、(日付などを)勝手に整形するのは邪魔。

Javaプログラマにとっての一番のメリットは、かなり小規模なアプリケーションであることだと思います。
機能が足りなければ、勝手に足しちゃいな！

DBConnect
=========

Java, databse client using jdbc.

Swing GUIによる、JDBCクライアントツールです。<br>
Javaプログラミング時に、JDBCのレスポンスを素のままで観察したかったことから作成を始め、あれこれ機能追加をしてたらこうなりました。<br>
以下のように思う方は使ってみてくださいな。<br>

<ul>
<li> DBMSによってクライアントツール違うのは、キーバインドなどが違って不便！</li>
<li> 正規のクライアントツールが、(日付などを)勝手に整形するのは邪魔。</li>
</ul>
<br>
Javaプログラマにとっての一番のメリットは、かなり小規模なアプリケーションであることだと思います。<br>
機能が足りなければ、勝手に足しちゃいな！<br>
